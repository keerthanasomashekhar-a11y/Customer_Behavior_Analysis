# Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior data using Python, Pandas, PostgreSQL, and SQL.

## Project Objectives
- Analyze customer purchasing behavior
- Identify customer segments
- Study sales and revenue trends
- Evaluate discount impact on purchases
- Perform product and category analysis
- Generate business insights using SQL queries

- Pandas
- PostgreSQL

- pgAdmin

- PostgreSQL Database Integration

- Product Rating Analysis
- Discount Usage Analysis
- Purchase Frequency Analysis
- SQL Query Practice and Analytics
